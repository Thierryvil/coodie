
# coodie

Plataforma para contratação de profissionais de tecnologia.


## Variáveis de Ambiente

Para rodar esse projeto, você vai precisar adicionar as seguintes variáveis de ambiente

Backend:

`SECRET_KEY`

`DATABASE_NAME`

`ALGORITHM`

`ACCESS_TOKEN_EXPIRE_MINUTES`


Frontend:

`NEXTAUTH_URL`

`NEXTAUTH_SECRET`

`COODIE_API_URL`
